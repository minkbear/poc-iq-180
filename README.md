## Overview

This repository contains the codebase for the MB-MVP project.

## Roadmap

See [ROADMAP.md](./ROADMAP.md) for the current milestone plan.

## Getting Started

Open `index.html` directly in a browser, or run a local server:

```bash
npm start   # serves on http://localhost:3000
```

## Features

- **Random Numbers** — displays N random single-digit numbers (count configurable 1–20)
- **Math Challenge** — random +/−/× problem with configurable max value; reveal answer on demand
- **Countdown Timer** — configurable 1–60 minutes; plays a sound on expiry
- **Settings panel** — all parameters adjustable without a reload
