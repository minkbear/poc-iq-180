# Kids Math Game — Getting Started

A quick guide for parents and teachers setting up the game for kids.

---

## Opening the Game

Visit the live game in any modern browser:

**https://minkbear.github.io/mb-mvp/**

No installation or login required. Works on desktop and tablet.

---

## How to Play

### 1. Start a Round

Click **Start** in the Countdown Timer card. The timer counts down from the configured duration. When it reaches zero, a short musical chime plays to signal the round is over.

### 2. Random Numbers

A row of coloured number cards is shown. These are the numbers the child will work with during the round. Click **New Numbers & Problem** at any time to get a fresh set.

### 3. Math Challenge

A single arithmetic problem (addition, subtraction, or multiplication) is displayed. The child works it out, then clicks **Show Answer** to check.

Click **New Numbers & Problem** to move to the next problem.

### 4. Pausing and Resetting

| Button    | Effect                                               |
|-----------|------------------------------------------------------|
| **Start** | Begins the countdown                                 |
| **Pause** | Freezes the timer where it is                        |
| **Reset** | Stops the timer and restores it to the full duration |

---

## Changing Settings

The **Settings** panel at the top of the page lets you customise each round before you start.

| Setting              | What it controls                          | Range   | Default |
|----------------------|-------------------------------------------|---------|---------|
| Number Count         | How many random number cards are shown    | 1–20    | 5       |
| Math Max Value       | Largest number used in math problems      | 2–999   | 99      |
| Timer Minutes        | Length of each round                      | 1–60    | 3       |

Changes take effect immediately — the numbers and problem refresh automatically and the timer resets to the new duration.

---

## What Happens When the Timer Expires?

The timer display turns red and a short three-note chime plays through the device speaker. The child cannot continue on the same timer — click **Reset** then **Start** to begin a new round.

> **Tip:** Some browsers require a user interaction (like pressing Start) before they allow audio. If the chime does not play, make sure the child clicked Start themselves rather than having the page auto-start.
